# echochoke
automatically capture your fails and send them to a discord webhook with custom roasts

### how to use
1. create a discord webhook in your server settings
2. paste the url into the mod settings in game
3. set your minimum percentage for reporting
4. every time you die above that percent the mod will post a screenshot and a roast to your discord

### custom roasts
you can find roasts.txt in your mod save folder to add your own messages. use {} to show where the percentage and level name should go"# EchoChoke"
