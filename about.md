# EchoChoke

Echochoke is a mod that broadcasts your skill issues to discord

Whenever you make a new best, the mod grabs a screenshot of your screen and sends it to a webhook of your choice (provided channel's webhook). It includes a random roast from a *customizable* list so your friends can witness the choke in real time!

Features
- automated discord webhook integration
- automatic screenshots on death
- customizable roast list via text file
- minimum percentage threshold to avoid spam
- + a LOT more
